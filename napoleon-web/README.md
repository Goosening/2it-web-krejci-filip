# Napoleon Bonaparte - One-Page Web

Vítejte v projektu moderní jednostránkové prezentace o Napoleonu Bonapartovi. Tento web kombinuje historickou estetikou s moderními webovými technologiemi pro vytvoření "wow" efektu.

## O projektu
Cílem byla vytvořit responzivní a vizuálně poutavou stránku, která provede návštěvníka životem a odkazem jednoho z největších vojevůdců historie. Design využívá královské barvy, textury pergamenu a dramatické olejomalby, vše oživené plynulými animacemi.

## Technologie a Knihovny
- **HTML5 & CSS3**: Sémantická struktura a moderní layouty (Flexbox/Grid).
- **JavaScript (ES6+)**: Logika interakcí.
- **GSAP (GreenSock Animation Platform)**: Použito pro pokročilé scroll-based animace (ScrollTrigger) a parallax efekty.
- **Google Fonts**: Písma 'Cinzel' (nadpisy) a 'Inter' (text).
- **Gemini AI**: Použito pro generování historických vizuálů.
- **Antigravity**: Použito pro pomoc s výtvorem.

## Návod na spuštění
Tento projekt nevyžaduje žádný build proces.
1. Stáhněte si repozitář.
2. Otevřete soubor `index.html` v libovolném moderním webovém prohlížeči.
3. Pro plný zážitek doporučujeme prohlížeč s podporou hardwarové akcelerace (Chrome, Firefox, Edge).

## Highlights (Na co jsem nejvíce hrdý)
- **Parallax Hero Sekce**: Úvodní obrazovka s efektem hloubky.
- **Interaktivní Časová Osa**: Historické milníky se objevují s plynulou animací při skrolování.
- **Atmosféra**: Kombinace generovaných historických vizuálů s moderním UI designem.
